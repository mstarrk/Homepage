Thank you for checking this website I made using HTML, CSS, JavaScript, and Bootstrap 5.


This is my solution for CS50's Problem Set 8, called Homepage. The challenge was to build a static website that adheres to the specifications provided using the languages and tools mentioned above.

I had the freedom to choose what the website should look like. I went with a showcase on CS50 itself, strongly focusing on CSS to make a visually appealing and responsive design.


Although I've mentioned many things about the course, I think it does not do justice to how good it is. If you are interested in Computer Science or are a developer, I strongly encourage you to try the course.


I've been doing programming for the last few years. My curiosity about the subject began with gamedev and Unity, which led me to university to learn the fundamentals of object-oriented programming and C#.


The university has given me a diverse range of knowledge and experience, including:

  • Discrete mathematics and statistics
  • Web development with PHP and SQL
  • Fullstack desktop applications using Windows Forms and C#


However, I still had one missing piece from the puzzle that CS50 helped me find: what's going on behind the curtains?


I had these questions because of the abstraction that I was not familiar with coming from a high-level programming language like C#.


Questions like:

  • What really is an array?
  • What makes an int?
  • Where are these things even defined?
  • How does hardware come into play?

And a million more that made me feel like an impostor.


Initially, I was very skeptical of the course because I already have some programming experience.

I thought it was going to be boring. But I immediately fell in love with the way the course was taught and the unbelievable production work it had.

I mistakenly thought it was going to be easy. And it was ... for the first two problem sets. Then Tideman happened. Needless to say, it was challenging.


Now that I'm just about to finish the course., I can say most of those doubts I had are now out of my mind.


Thank you, CS50, for helping me grow into a better problem solver. üíù
