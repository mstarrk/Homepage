What is CS50?

An introduction to the intellectual enterprises of computer science and the art of programming.

An entry-level course taught by David J. Malan, CS50x teaches students how to think algorithmically and solve problems efficiently.

Topics include abstraction, algorithms, data structures, encapsulation, resource management, security, software engineering, and web development.

Languages include C, Python, SQL, and JavaScript plus CSS and HTML.

Problem sets inspired by real-world domains of biology, cryptography, finance, forensics, and gaming.

Students who earn a satisfactory score on 9 problem sets (i.e., programming assignments) and a final project are eligible for a certificate.

What will you learn?

Problem solving with C

Dive into the C programming language, a low-level language that will show you how everything functions behind the curtains. Over the course of 5 lectures and a variety of different sets of problems and tasks, you will develop a solid programming foundation that will include algorithms, data structures and memory management.

Python

Then we'll switch to Python: a high-level programming language designed for every purpose. While working on very interesting problem sets, you will add Python to your toolkit to solve many different problems. You will learn how to work with data, file system and use Python to develop applications, automate processes, and even build your own websites and APIs.

SQL

SQL (Structure Query Language) is an in-demand skill. Knowledge of SQL is a valuable skill in many industries. Developers, Business Analysts, and Product Managers are just a few of the many professions that rely on SQL for data retrieval. In this course you'll learn how to use it to manage data easier and to use it in client-server operations.

Web Development

Last but not least, we will work on web development by learning languages, HTML, CSS and JavaScript to develop web applications. Those three languages are the building blocks of the Internet as we know it. We will also learn how to leverage our newly acquired tool, Python, in the development of server-side applications.
